## 🏗️ Part 1 — Building a Proper Script (The Right Way)

Before we jump into cool scripts, let's learn how to **structure** a DevOps script properly. Every script you write should have these pieces:

### The Script Template (Use This for Everything)

```python
#!/usr/bin/env python3
"""
Script Name: my_automation.py
Description: What this script does (one line).
Author: Your Name
Usage: python3 my_automation.py --arg1 value1 --arg2 value2
"""

# ═══════════════════════════════════════
# IMPORTS
# ═══════════════════════════════════════
import argparse
import logging
import sys
import os
from datetime import datetime

# ═══════════════════════════════════════
# LOGGING SETUP
# ═══════════════════════════════════════
logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(message)s",
    datefmt="%Y-%m-%d %H:%M:%S",
    handlers=[
        logging.StreamHandler(sys.stdout),                          # Print to console
        logging.FileHandler(f"logs/my_script_{datetime.now().strftime('%Y%m%d')}.log")  # Save to file
    ]
)
logger = logging.getLogger(__name__)

# ═══════════════════════════════════════
# FUNCTIONS
# ═══════════════════════════════════════
def main(args):
    """Main function — your script logic goes here."""
    logger.info("Script started")
    
    # Your automation code here
    
    logger.info("Script completed successfully")

# ═══════════════════════════════════════
# ARGUMENT PARSING
# ═══════════════════════════════════════
def parse_args():
    """Parse command-line arguments."""
    parser = argparse.ArgumentParser(description="My automation script")
    parser.add_argument("--env", required=True, choices=["dev", "staging", "prod"], help="Target environment")
    parser.add_argument("--dry-run", action="store_true", help="Show what would happen without doing it")
    parser.add_argument("--verbose", "-v", action="store_true", help="Enable debug logging")
    return parser.parse_args()

# ═══════════════════════════════════════
# ENTRY POINT
# ═══════════════════════════════════════
if __name__ == "__main__":
    args = parse_args()
    
    if args.verbose:
        logging.getLogger().setLevel(logging.DEBUG)
    
    try:
        main(args)
    except KeyboardInterrupt:
        logger.warning("Script interrupted by user (Ctrl+C)")
        sys.exit(130)
    except Exception as e:
        logger.error(f"Script failed: {e}", exc_info=True)
        sys.exit(1)
```

**Let's break down WHY each piece matters:**

| Piece | Why It Matters |
|-------|---------------|
| `#!/usr/bin/env python3` | Makes the script executable on Linux directly: `./script.py` |
| Docstring at top | Anyone can understand what the script does |
| `logging` (not `print`) | Logs have timestamps, levels, can write to files |
| `argparse` | Clean command-line arguments, auto-generates `--help` |
| `if __name__ == "__main__"` | Allows importing functions without running the script |
| `try/except` | Catches errors and exits with proper error codes |
| `--dry-run` flag | Lets you test without actually doing anything (very important!) |

### Why logging Instead of print?

```python
# ❌ Using print — no timestamp, no level, no file output
print("Deploying to production...")
print("ERROR: Deployment failed!")

# ✅ Using logging — timestamps, levels, automatically saved to file
logger.info("Deploying to production...")
logger.error("Deployment failed!")
```

**Output with logging:**
```
2024-01-15 14:30:00 [INFO] Deploying to production...
2024-01-15 14:30:05 [ERROR] Deployment failed!
```

**Logging levels (from least to most serious):**

| Level | When To Use | Example |
|-------|------------|---------|
| `DEBUG` | Detailed info for debugging | `logger.debug(f"Checking server {ip}")` |
| `INFO` | Normal operations | `logger.info("Deployment started")` |
| `WARNING` | Something unexpected but not broken | `logger.warning("Disk usage at 85%")` |
| `ERROR` | Something failed | `logger.error("Could not connect to DB")` |
| `CRITICAL` | Everything is broken | `logger.critical("All servers down!")` |

## 🖥️ Part 2 — Server Health Check Script

This script checks the health of multiple servers — disk space, memory, CPU, and running services:

```python
#!/usr/bin/env python3
"""
Server Health Check Script
Checks disk, memory, CPU, and service status on remote servers via SSH.
Usage: python3 health_check.py --servers servers.txt --threshold 80
"""

import subprocess
import argparse
import logging
import sys
import json
from datetime import datetime

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(message)s"
)
logger = logging.getLogger(__name__)


def run_ssh_command(host, command, user="ubuntu"):
    """
    Run a command on a remote server via SSH.
    Returns the command output as a string.
    """
    ssh_command = [
        "ssh",
        "-o", "StrictHostKeyChecking=no",    # Don't ask about host fingerprints
        "-o", "ConnectTimeout=10",            # Timeout after 10 seconds
        f"{user}@{host}",
        command
    ]
    
    try:
        result = subprocess.run(
            ssh_command,
            capture_output=True,
            text=True,
            timeout=30
        )
        
        if result.returncode != 0:
            logger.error(f"Command failed on {host}: {result.stderr.strip()}")
            return None
        
        return result.stdout.strip()
        
    except subprocess.TimeoutExpired:
        logger.error(f"SSH command timed out on {host}")
        return None
    except FileNotFoundError:
        logger.error("SSH client not found. Is OpenSSH installed?")
        return None


def check_disk_usage(host, threshold=80):
    """Check disk usage on a remote server."""
    output = run_ssh_command(host, "df -h / | tail -1 | awk '{print $5}' | tr -d '%'")
    
    if output is None:
        return {"status": "ERROR", "message": "Could not check disk"}
    
    usage = int(output)
    status = "CRITICAL" if usage >= threshold else "OK"
    
    return {
        "metric": "Disk Usage",
        "value": f"{usage}%",
        "threshold": f"{threshold}%",
        "status": status
    }


def check_memory_usage(host, threshold=80):
    """Check memory usage on a remote server."""
    output = run_ssh_command(host, "free | grep Mem | awk '{printf \"%.0f\", $3/$2 * 100}'")
    
    if output is None:
        return {"status": "ERROR", "message": "Could not check memory"}
    
    usage = int(output)
    status = "CRITICAL" if usage >= threshold else "OK"
    
    return {
        "metric": "Memory Usage",
        "value": f"{usage}%",
        "threshold": f"{threshold}%",
        "status": status
    }


def check_cpu_load(host, threshold=80):
    """Check CPU load average on a remote server."""
    output = run_ssh_command(host, "top -bn1 | grep 'Cpu(s)' | awk '{print $2}' | cut -d'.' -f1")
    
    if output is None:
        return {"status": "ERROR", "message": "Could not check CPU"}
    
    usage = int(output)
    status = "CRITICAL" if usage >= threshold else "OK"
    
    return {
        "metric": "CPU Usage",
        "value": f"{usage}%",
        "threshold": f"{threshold}%",
        "status": status
    }


def check_service_status(host, services):
    """Check if specific services are running on a remote server."""
    results = []
    
    for service in services:
        output = run_ssh_command(host, f"systemctl is-active {service}")
        
        if output == "active":
            results.append({"service": service, "status": "RUNNING ✅"})
        else:
            results.append({"service": service, "status": "DOWN ❌"})
    
    return results


def generate_report(all_results):
    """Generate a formatted health check report."""
    
    print("\n" + "=" * 70)
    print(f"  SERVER HEALTH CHECK REPORT — {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print("=" * 70)
    
    critical_count = 0
    
    for server, checks in all_results.items():
        print(f"\n🖥️  Server: {server}")
        print("-" * 50)
        
        for check in checks:
            if isinstance(check, dict) and "metric" in check:
                icon = "❌" if check["status"] == "CRITICAL" else "✅"
                print(f"  {icon} {check['metric']:15} : {check['value']:8} (threshold: {check['threshold']})")
                if check["status"] == "CRITICAL":
                    critical_count += 1
            elif isinstance(check, dict) and "service" in check:
                print(f"  {check['status']:>8} — {check['service']}")
    
    print("\n" + "=" * 70)
    if critical_count > 0:
        print(f"  ⚠️  {critical_count} CRITICAL issues found!")
    else:
        print("  ✅ All servers are healthy!")
    print("=" * 70 + "\n")
    
    return critical_count


def main():
    parser = argparse.ArgumentParser(description="Server Health Check Script")
    parser.add_argument("--servers", required=True, help="File with server IPs (one per line)")
    parser.add_argument("--threshold", type=int, default=80, help="Alert threshold percentage")
    parser.add_argument("--services", nargs="+", default=["nginx", "docker"], help="Services to check")
    args = parser.parse_args()
    
    # Read server list
    try:
        with open(args.servers, "r") as f:
            servers = [line.strip() for line in f if line.strip() and not line.startswith("#")]
    except FileNotFoundError:
        logger.error(f"Server file not found: {args.servers}")
        sys.exit(1)
    
    logger.info(f"Checking {len(servers)} servers with threshold {args.threshold}%")
    
    all_results = {}
    
    for server in servers:
        logger.info(f"Checking {server}...")
        checks = []
        
        checks.append(check_disk_usage(server, args.threshold))
        checks.append(check_memory_usage(server, args.threshold))
        checks.append(check_cpu_load(server, args.threshold))
        checks.extend(check_service_status(server, args.services))
        
        all_results[server] = checks
    
    critical = generate_report(all_results)
    
    # Exit with error code if there are critical issues
    sys.exit(1 if critical > 0 else 0)


if __name__ == "__main__":
    main()
```

## 📁 Part 3 — Log Analyzer Script

This script analyzes log files to find error patterns, count occurrences, and generate a summary:

```python
#!/usr/bin/env python3
"""
Log Analyzer Script
Analyzes log files for errors, warnings, and patterns.
Usage: python3 log_analyzer.py --logfile /var/log/app.log --top 10
"""

import re
import argparse
from collections import Counter, defaultdict
from datetime import datetime


def analyze_log_file(logfile, top_n=10):
    """Analyze a log file and return statistics."""
    
    # Counters
    level_counts = Counter()           # Count of each log level
    error_messages = Counter()         # Count of each unique error message
    errors_by_hour = defaultdict(int)  # Errors grouped by hour
    ip_addresses = Counter()           # Count of requests by IP
    status_codes = Counter()           # Count of HTTP status codes
    total_lines = 0
    
    # Common log patterns (regex)
    # Matches: 2024-01-15 14:30:00 [ERROR] Something went wrong
    log_pattern = re.compile(
        r'(\d{4}-\d{2}-\d{2}\s\d{2}:\d{2}:\d{2})\s*\[(\w+)\]\s*(.*)'
    )
    
    # Matches: 192.168.1.1 - - [15/Jan/2024:14:30:00] "GET /api/health" 200
    access_log_pattern = re.compile(
        r'(\d+\.\d+\.\d+\.\d+).*?"(\w+)\s+(\S+).*?"\s+(\d{3})'
    )
    
    print(f"📊 Analyzing: {logfile}")
    print("-" * 60)
    
    try:
        with open(logfile, 'r', encoding='utf-8', errors='replace') as f:
            for line in f:
                total_lines += 1
                
                # Try application log format
                match = log_pattern.match(line)
                if match:
                    timestamp, level, message = match.groups()
                    level_counts[level] += 1
                    
                    if level in ("ERROR", "CRITICAL", "FATAL"):
                        error_messages[message.strip()] += 1
                        hour = timestamp[:13]  # "2024-01-15 14"
                        errors_by_hour[hour] += 1
                    continue
                
                # Try access log format
                match = access_log_pattern.search(line)
                if match:
                    ip, method, path, status = match.groups()
                    ip_addresses[ip] += 1
                    status_codes[status] += 1
                    
                    if status.startswith(('4', '5')):
                        error_messages[f"{status} {method} {path}"] += 1
                        
    except FileNotFoundError:
        print(f"❌ File not found: {logfile}")
        return
    except PermissionError:
        print(f"❌ Permission denied: {logfile}")
        return
    
    # ═══════════════════════════════════════
    # PRINT REPORT
    # ═══════════════════════════════════════
    print(f"\n{'='*60}")
    print(f"  LOG ANALYSIS REPORT")
    print(f"  File: {logfile}")
    print(f"  Total Lines: {total_lines:,}")
    print(f"{'='*60}")
    
    # Log Level Distribution
    if level_counts:
        print(f"\n📈 Log Level Distribution:")
        for level, count in level_counts.most_common():
            percentage = (count / total_lines) * 100
            bar = "█" * int(percentage / 2)
            print(f"  {level:10} : {count:>8,} ({percentage:5.1f}%) {bar}")
    
    # HTTP Status Codes
    if status_codes:
        print(f"\n🌐 HTTP Status Codes:")
        for code, count in status_codes.most_common():
            icon = "✅" if code.startswith('2') else "⚠️" if code.startswith('3') else "❌"
            print(f"  {icon} {code} : {count:>8,}")
    
    # Top Errors
    if error_messages:
        print(f"\n🔥 Top {top_n} Errors:")
        for i, (msg, count) in enumerate(error_messages.most_common(top_n), 1):
            # Truncate long messages
            display_msg = msg[:70] + "..." if len(msg) > 70 else msg
            print(f"  {i:>2}. [{count:>5,}x] {display_msg}")
    
    # Errors by Hour
    if errors_by_hour:
        print(f"\n⏰ Errors by Hour:")
        for hour, count in sorted(errors_by_hour.items()):
            bar = "█" * min(count, 50)
            print(f"  {hour} : {count:>5,} {bar}")
    
    # Top IPs
    if ip_addresses:
        print(f"\n🌍 Top {top_n} IP Addresses:")
        for ip, count in ip_addresses.most_common(top_n):
            print(f"  {ip:>18} : {count:>8,} requests")
    
    print(f"\n{'='*60}")
    
    # Return summary for further processing
    return {
        "total_lines": total_lines,
        "level_counts": dict(level_counts),
        "top_errors": error_messages.most_common(top_n),
        "errors_by_hour": dict(errors_by_hour)
    }


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Analyze log files")
    parser.add_argument("--logfile", required=True, help="Path to log file")
    parser.add_argument("--top", type=int, default=10, help="Number of top errors to show")
    args = parser.parse_args()
    
    analyze_log_file(args.logfile, args.top)
```

## 🔄 Part 4 — Automated Backup Script

This script creates automated backups of directories and databases, compresses them, and optionally uploads to S3:

```python
#!/usr/bin/env python3
"""
Automated Backup Script
Creates compressed backups of directories and databases.
Supports local storage and S3 upload.
Usage: python3 backup.py --config backup_config.json
"""

import os
import subprocess
import tarfile
import logging
import argparse
import json
import sys
from datetime import datetime, timedelta
from pathlib import Path

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(message)s"
)
logger = logging.getLogger(__name__)


def create_directory_backup(source_dir, backup_dir):
    """
    Create a compressed tar.gz backup of a directory.
    Returns the path to the backup file.
    """
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    dir_name = os.path.basename(source_dir.rstrip('/'))
    backup_filename = f"{dir_name}_backup_{timestamp}.tar.gz"
    backup_path = os.path.join(backup_dir, backup_filename)
    
    logger.info(f"Backing up directory: {source_dir}")
    
    if not os.path.exists(source_dir):
        logger.error(f"Source directory not found: {source_dir}")
        return None
    
    # Create backup directory if it doesn't exist
    os.makedirs(backup_dir, exist_ok=True)
    
    try:
        with tarfile.open(backup_path, "w:gz") as tar:
            tar.add(source_dir, arcname=dir_name)
        
        size_mb = os.path.getsize(backup_path) / (1024 * 1024)
        logger.info(f"✅ Backup created: {backup_path} ({size_mb:.2f} MB)")
        return backup_path
        
    except PermissionError:
        logger.error(f"Permission denied when backing up {source_dir}")
        return None
    except Exception as e:
        logger.error(f"Backup failed: {e}")
        return None


def create_mysql_backup(host, user, password, database, backup_dir):
    """
    Create a MySQL database backup using mysqldump.
    Returns the path to the backup file.
    """
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_filename = f"{database}_db_backup_{timestamp}.sql.gz"
    backup_path = os.path.join(backup_dir, backup_filename)
    
    logger.info(f"Backing up MySQL database: {database}@{host}")
    
    os.makedirs(backup_dir, exist_ok=True)
    
    try:
        # mysqldump command — pipe through gzip for compression
        dump_command = [
            "mysqldump",
            f"--host={host}",
            f"--user={user}",
            f"--password={password}",
            "--single-transaction",   # Consistent snapshot for InnoDB
            "--routines",             # Include stored procedures
            "--triggers",             # Include triggers
            database
        ]
        
        with open(backup_path, 'wb') as f:
            dump_process = subprocess.Popen(dump_command, stdout=subprocess.PIPE, stderr=subprocess.PIPE)
            gzip_process = subprocess.Popen(["gzip"], stdin=dump_process.stdout, stdout=f)
            
            dump_process.stdout.close()
            gzip_process.communicate()
            
            if dump_process.wait() != 0:
                stderr = dump_process.stderr.read().decode()
                logger.error(f"mysqldump failed: {stderr}")
                return None
        
        size_mb = os.path.getsize(backup_path) / (1024 * 1024)
        logger.info(f"✅ DB backup created: {backup_path} ({size_mb:.2f} MB)")
        return backup_path
        
    except FileNotFoundError:
        logger.error("mysqldump not found. Is MySQL client installed?")
        return None
    except Exception as e:
        logger.error(f"Database backup failed: {e}")
        return None


def create_postgresql_backup(host, user, database, backup_dir, port=5432):
    """Create a PostgreSQL database backup using pg_dump."""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    backup_filename = f"{database}_pg_backup_{timestamp}.sql.gz"
    backup_path = os.path.join(backup_dir, backup_filename)
    
    logger.info(f"Backing up PostgreSQL database: {database}@{host}")
    
    os.makedirs(backup_dir, exist_ok=True)
    
    try:
        dump_command = [
            "pg_dump",
            f"--host={host}",
            f"--port={port}",
            f"--username={user}",
            "--format=custom",    # Custom format for pg_restore
            f"--file={backup_path}",
            database
        ]
        
        result = subprocess.run(dump_command, capture_output=True, text=True, timeout=600)
        
        if result.returncode != 0:
            logger.error(f"pg_dump failed: {result.stderr}")
            return None
        
        size_mb = os.path.getsize(backup_path) / (1024 * 1024)
        logger.info(f"✅ PG backup created: {backup_path} ({size_mb:.2f} MB)")
        return backup_path
        
    except subprocess.TimeoutExpired:
        logger.error("pg_dump timed out after 600 seconds")
        return None
    except FileNotFoundError:
        logger.error("pg_dump not found. Is PostgreSQL client installed?")
        return None


def upload_to_s3(file_path, bucket, s3_prefix="backups/"):
    """Upload a backup file to AWS S3."""
    try:
        import boto3
        
        s3 = boto3.client('s3')
        filename = os.path.basename(file_path)
        s3_key = f"{s3_prefix}{filename}"
        
        logger.info(f"Uploading to s3://{bucket}/{s3_key}...")
        
        # Upload with progress callback
        file_size = os.path.getsize(file_path)
        
        s3.upload_file(
            file_path,
            bucket,
            s3_key,
            ExtraArgs={'StorageClass': 'STANDARD_IA'}  # Cheaper storage for backups
        )
        
        logger.info(f"✅ Uploaded to s3://{bucket}/{s3_key}")
        return True
        
    except ImportError:
        logger.error("boto3 not installed. Run: pip install boto3")
        return False
    except Exception as e:
        logger.error(f"S3 upload failed: {e}")
        return False


def cleanup_old_backups(backup_dir, retention_days=30):
    """Delete local backup files older than retention_days."""
    
    cutoff = datetime.now() - timedelta(days=retention_days)
    deleted = 0
    freed_space = 0
    
    logger.info(f"Cleaning up backups older than {retention_days} days in {backup_dir}")
    
    for filepath in Path(backup_dir).glob("*_backup_*"):
        if filepath.is_file():
            file_time = datetime.fromtimestamp(filepath.stat().st_mtime)
            if file_time < cutoff:
                size = filepath.stat().st_size
                filepath.unlink()
                deleted += 1
                freed_space += size
                logger.info(f"  🗑️  Deleted: {filepath.name}")
    
    freed_mb = freed_space / (1024 * 1024)
    logger.info(f"✅ Cleaned up {deleted} old backups, freed {freed_mb:.2f} MB")

    # Backup PostgreSQL databases
```

## 🚀 Part 5 — Deployment Automation Script

This is the big one — a script that deploys your application to multiple servers:

```python
#!/usr/bin/env python3
"""
Deployment Automation Script
Deploys application to servers with rollback support.
Usage: python3 deploy.py --env production --version v2.3.1
"""

import subprocess
import argparse
import logging
import sys
import json
import os
from datetime import datetime

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(message)s"
)
logger = logging.getLogger(__name__)


class Deployer:
    """Handles deployment to multiple servers."""
    
    def __init__(self, config, version, dry_run=False):
        self.config = config
        self.version = version
        self.dry_run = dry_run
        self.deploy_log = []
        self.deployed_servers = []  # Track for rollback
    
    def run_remote(self, host, command, user="ubuntu"):
        """Run a command on a remote server via SSH."""
        if self.dry_run:
            logger.info(f"  [DRY RUN] {host}: {command}")
            return True, "dry-run"
        
        ssh_command = [
            "ssh", "-o", "StrictHostKeyChecking=no",
            "-o", "ConnectTimeout=10",
            f"{user}@{host}", command
        ]
        
        try:
            result = subprocess.run(
                ssh_command, capture_output=True, text=True, timeout=120
            )
            
            if result.returncode != 0:
                logger.error(f"  Command failed on {host}: {result.stderr.strip()}")
                return False, result.stderr.strip()
            
            return True, result.stdout.strip()
            
        except subprocess.TimeoutExpired:
            return False, "Command timed out"
        except Exception as e:
            return False, str(e)
    
    def health_check(self, host, port=80, path="/health"):
        """Check if the application is healthy after deployment."""
        import requests
        
        url = f"http://{host}:{port}{path}"
        
        try:
            response = requests.get(url, timeout=10)
            if response.status_code == 200:
                logger.info(f"  ✅ Health check passed: {host}")
                return True
            else:
                logger.error(f"  ❌ Health check failed: {host} — Status {response.status_code}")
                return False
        except Exception as e:
            logger.error(f"  ❌ Health check failed: {host} — {e}")
            return False
    
    def deploy_to_server(self, host, user="ubuntu"):
        """Deploy to a single server."""
        app_dir = self.config["app_dir"]
        service_name = self.config["service_name"]
        
        logger.info(f"\n🚀 Deploying {self.version} to {host}...")
        
        steps = [
            # Step 1: Create a backup of current version
            (
                "Backing up current version",
                f"cd {app_dir} && cp -r . /tmp/app_backup_$(date +%Y%m%d_%H%M%S)"
            ),
            # Step 2: Pull latest code
            (
                "Pulling latest code",
                f"cd {app_dir} && git fetch --all && git checkout {self.version}"
            ),
            # Step 3: Install dependencies
            (
                "Installing dependencies",
                f"cd {app_dir} && pip install -r requirements.txt"
            ),
            # Step 4: Run database migrations (if needed)
            (
                "Running migrations",
                f"cd {app_dir} && python manage.py migrate --noinput"
            ),
            # Step 5: Restart the service
            (
                "Restarting service",
                f"sudo systemctl restart {service_name}"
            ),
            # Step 6: Wait for service to start
            (
                "Waiting for service",
                f"sleep 5 && sudo systemctl is-active {service_name}"
            )
        ]
        
        for step_name, command in steps:
            logger.info(f"  📋 {step_name}...")
            success, output = self.run_remote(host, command, user)
            
            if not success:
                logger.error(f"  ❌ Failed at step: {step_name}")
                self.deploy_log.append({
                    "host": host,
                    "step": step_name,
                    "status": "FAILED",
                    "error": output
                })
                return False
            
            self.deploy_log.append({
                "host": host,
                "step": step_name,
                "status": "OK"
            })
        
        self.deployed_servers.append(host)
        return True
    
    def rollback_server(self, host, user="ubuntu"):
        """Rollback a server to the previous version."""
        app_dir = self.config["app_dir"]
        service_name = self.config["service_name"]
        
        logger.warning(f"⏪ Rolling back {host}...")
        
        # Find the latest backup
        success, backup_dir = self.run_remote(
            host,
            "ls -td /tmp/app_backup_* 2>/dev/null | head -1",
            user
        )
        
        if success and backup_dir:
            self.run_remote(host, f"rm -rf {app_dir}/* && cp -r {backup_dir}/* {app_dir}/", user)
            self.run_remote(host, f"sudo systemctl restart {service_name}", user)
            logger.info(f"  ✅ Rollback completed on {host}")
        else:
            logger.error(f"  ❌ No backup found on {host} — manual intervention needed!")
    
    def deploy(self):
        """Deploy to all servers."""
        servers = self.config["servers"]
        user = self.config.get("ssh_user", "ubuntu")
        
        logger.info("=" * 60)
        logger.info(f"  DEPLOYMENT STARTED")
        logger.info(f"  Version: {self.version}")
        logger.info(f"  Servers: {len(servers)}")
        logger.info(f"  Dry Run: {self.dry_run}")
        logger.info("=" * 60)
        
        failed_servers = []
        
        for server in servers:
            success = self.deploy_to_server(server, user)
            
            if success:
                # Run health check
                if not self.dry_run:
                    health_port = self.config.get("health_port", 80)
                    health_path = self.config.get("health_path", "/health")
                    
                    if not self.health_check(server, health_port, health_path):
                        logger.error(f"Health check failed! Rolling back {server}...")
                        self.rollback_server(server, user)
                        failed_servers.append(server)
            else:
                failed_servers.append(server)
                
                # Should we continue or rollback everything?
                if self.config.get("rollback_on_failure", True):
                    logger.error("Deployment failed! Rolling back all deployed servers...")
                    for deployed_host in self.deployed_servers:
                        self.rollback_server(deployed_host, user)
                    break
        
        # Print summary
        logger.info("\n" + "=" * 60)
        logger.info(f"  DEPLOYMENT SUMMARY")
        logger.info(f"  ✅ Successful: {len(self.deployed_servers) - len(failed_servers)}")
        logger.info(f"  ❌ Failed: {len(failed_servers)}")
        if failed_servers:
            logger.info(f"  Failed servers: {', '.join(failed_servers)}")
        logger.info("=" * 60)
        
        return len(failed_servers) == 0


def main():
    parser = argparse.ArgumentParser(description="Deployment Automation")
    parser.add_argument("--config", default="deploy_config.json", help="Deployment config file")
    parser.add_argument("--version", required=True, help="Version to deploy (e.g., v2.3.1)")
    parser.add_argument("--dry-run", action="store_true", help="Show steps without executing")
    args = parser.parse_args()
    
    # Load config
    try:
        with open(args.config, 'r') as f:
            config = json.load(f)
    except FileNotFoundError:
        logger.error(f"Config not found: {args.config}")
        sys.exit(1)
    
    deployer = Deployer(config, args.version, args.dry_run)
    success = deployer.deploy()
    
    sys.exit(0 if success else 1)


if __name__ == "__main__":
    main()
```

**Deploy config — `deploy_config.json`:**
```json
{

    "service_name": "myapp",
    "ssh_user": "ubuntu",
    "health_port": 8080,
    "health_path": "/api/health",
    "rollback_on_failure": true,
    "servers": [
        "192.168.1.10",
        "192.168.1.11",
        "192.168.1.12"
    ]
}
```

## 📊 Part 6 — System Monitoring & Alert Script

```python
#!/usr/bin/env python3
"""
System Monitor & Alert Script
Monitors CPU, memory, disk, and sends alerts when thresholds are exceeded.
Usage: python3 monitor.py --config monitor_config.json
"""

import os
import subprocess
import logging
import json
import sys
import time
from datetime import datetime

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(message)s"
)
logger = logging.getLogger(__name__)


def get_disk_usage(path="/"):
    """Get disk usage percentage for a given path."""
    stat = os.statvfs(path)
    total = stat.f_blocks * stat.f_frsize
    free = stat.f_bfree * stat.f_frsize
    used = total - free
    percentage = (used / total) * 100
    
    return {
        "total_gb": total / (1024 ** 3),
        "used_gb": used / (1024 ** 3),
        "free_gb": free / (1024 ** 3),
        "percentage": round(percentage, 1)
    }


def get_memory_usage():
    """Get memory usage from /proc/meminfo."""
    with open('/proc/meminfo', 'r') as f:
        lines = f.readlines()
    
    mem_info = {}
    for line in lines:
        parts = line.split()
        key = parts[0].rstrip(':')
        value = int(parts[1])  # in KB
        mem_info[key] = value
    
    total = mem_info['MemTotal']
    available = mem_info['MemAvailable']
    used = total - available
    percentage = (used / total) * 100
    
    return {
        "total_gb": total / (1024 * 1024),
        "used_gb": used / (1024 * 1024),
        "available_gb": available / (1024 * 1024),
        "percentage": round(percentage, 1)
    }


def get_cpu_usage():
    """Get CPU usage percentage."""
    # Read /proc/stat twice with a 1-second gap
    def read_cpu_stat():
        with open('/proc/stat', 'r') as f:
            line = f.readline()
        parts = line.split()
        return [int(x) for x in parts[1:]]
    
    stat1 = read_cpu_stat()
    time.sleep(1)
    stat2 = read_cpu_stat()
    
    # Calculate delta
    delta = [s2 - s1 for s1, s2 in zip(stat1, stat2)]
    total = sum(delta)
    idle = delta[3]
    usage = ((total - idle) / total) * 100
    
    return round(usage, 1)


def get_top_processes(n=5):
    """Get top N processes by CPU and memory usage."""
    result = subprocess.run(
        ["ps", "aux", "--sort=-%cpu"],
        capture_output=True, text=True
    )
    
    processes = []
    for line in result.stdout.strip().split('\n')[1:n+1]:
        parts = line.split(None, 10)
        processes.append({
            "user": parts[0],
            "pid": parts[1],
            "cpu": parts[2],
            "mem": parts[3],
            "command": parts[10][:50] if len(parts) > 10 else "N/A"
        })
    
    return processes


def send_alert(message, webhook_url=None):
    """Send an alert via Slack webhook."""
    logger.warning(f"🚨 ALERT: {message}")
    
    if webhook_url:
        import requests
        
        payload = {
            "text": f"🚨 *SERVER ALERT*\n{message}\n_Server: {os.uname().nodename}_\n_Time: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}_"
        }
        
        try:
            response = requests.post(webhook_url, json=payload, timeout=10)
            if response.status_code == 200:
                logger.info("Alert sent to Slack ✅")
            else:
                logger.error(f"Failed to send Slack alert: {response.status_code}")
        except Exception as e:
            logger.error(f"Failed to send alert: {e}")


def monitor_once(config):
    """Run one monitoring cycle."""
    alerts = []
    thresholds = config.get("thresholds", {})
    
    # Check Disk
    disk = get_disk_usage()
    disk_threshold = thresholds.get("disk", 80)
    if disk["percentage"] >= disk_threshold:
        alerts.append(f"Disk usage at {disk['percentage']}% (threshold: {disk_threshold}%)")
    
    # Check Memory
    memory = get_memory_usage()
    mem_threshold = thresholds.get("memory", 80)
    if memory["percentage"] >= mem_threshold:
        alerts.append(f"Memory usage at {memory['percentage']}% (threshold: {mem_threshold}%)")
    
    # Check CPU
    cpu = get_cpu_usage()
    cpu_threshold = thresholds.get("cpu", 80)
    if cpu >= cpu_threshold:
        alerts.append(f"CPU usage at {cpu}% (threshold: {cpu_threshold}%)")
    
    # Print status
    print(f"\n{'='*50}")
    print(f"  System Status — {datetime.now().strftime('%H:%M:%S')}")
    print(f"{'='*50}")
    print(f"  CPU:    {cpu:5.1f}%  {'🔥' if cpu >= cpu_threshold else '✅'}")
    print(f"  Memory: {memory['percentage']:5.1f}%  ({memory['used_gb']:.1f}/{memory['total_gb']:.1f} GB)  {'🔥' if memory['percentage'] >= mem_threshold else '✅'}")
    print(f"  Disk:   {disk['percentage']:5.1f}%  ({disk['used_gb']:.1f}/{disk['total_gb']:.1f} GB)  {'🔥' if disk['percentage'] >= disk_threshold else '✅'}")
    
    # Send alerts
    if alerts:
        alert_message = "\n".join(alerts)
        send_alert(alert_message, config.get("slack_webhook"))
    
    return len(alerts)


def main():
    parser = argparse.ArgumentParser(description="System Monitor")
    parser.add_argument("--config", default="monitor_config.json", help="Config file")
    parser.add_argument("--once", action="store_true", help="Run once and exit")
    parser.add_argument("--interval", type=int, default=60, help="Check interval in seconds")
    args = parser.parse_args()
    
    # Load config
    try:
        with open(args.config, 'r') as f:

    except FileNotFoundError:

        config = {"thresholds": {"cpu": 80, "memory": 80, "disk": 80}}
    
    if args.once:

        sys.exit(1 if alerts > 0 else 0)
    
    # Continuous monitoring
    logger.info(f"Starting continuous monitoring (interval: {args.interval}s)")
    logger.info("Press Ctrl+C to stop")
    
    try:
        while True:
            monitor_once(config)
            time.sleep(args.interval)
    except KeyboardInterrupt:
        logger.info("Monitoring stopped by user")


if __name__ == "__main__":
    main()
```

```python
#!/usr/bin/env python3
"""
Config File Manager
Generates environment-specific config files from templates.
Usage: python3 config_manager.py --template app.conf.j2 --env production
"""

import argparse
import json
import os
import sys
import re
import logging
from pathlib import Path
from datetime import datetime

logging.basicConfig(level=logging.INFO, format="%(asctime)s [%(levelname)s] %(message)s")
logger = logging.getLogger(__name__)


def simple_template_render(template_str, variables):
    """
    Simple template renderer — replaces {{ variable_name }} with values.
    No Jinja2 dependency needed!
    """
    def replace_var(match):
        var_name = match.group(1).strip()
        if var_name in variables:
            return str(variables[var_name])
        else:
            logger.warning(f"Variable not found: {var_name}")
            return match.group(0)  # Leave as-is if not found
    
    # Replace {{ variable_name }} patterns
    result = re.sub(r'\{\{\s*(\w+)\s*\}\}', replace_var, template_str)
    return result


def load_env_variables(env_file):
    """Load variables from a JSON environment file."""
    try:
        with open(env_file, 'r') as f:
            return json.load(f)
    except FileNotFoundError:
        logger.error(f"Environment file not found: {env_file}")
        sys.exit(1)
    except json.JSONDecodeError as e:
        logger.error(f"Invalid JSON in {env_file}: {e}")
        sys.exit(1)


def generate_config(template_path, env_name, env_dir="environments", output_dir="generated"):
    """Generate a config file from a template and environment variables."""
    
    # Load template
    try:
        with open(template_path, 'r') as f:
            template = f.read()
    except FileNotFoundError:
        logger.error(f"Template not found: {template_path}")
        return None
    
    # Load environment variables
    env_file = os.path.join(env_dir, f"{env_name}.json")
    variables = load_env_variables(env_file)
    
    # Add automatic variables
    variables["generated_at"] = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    variables["environment"] = env_name
    
    # Render template
    rendered = simple_template_render(template, variables)
    
    # Save output
    os.makedirs(output_dir, exist_ok=True)
    template_name = os.path.basename(template_path)
    output_name = template_name.replace('.j2', '').replace('.template', '')
    output_path = os.path.join(output_dir, f"{env_name}_{output_name}")
    
    with open(output_path, 'w') as f:
        f.write(rendered)
    
    logger.info(f"✅ Generated: {output_path}")
    return output_path


# ═══════════════════════════════════════

# Run: python3 config_manager.py --template templates/nginx.conf.template --env production
# Output: generated/production_nginx.conf


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Config File Manager")
    parser.add_argument("--template", required=True, help="Path to template file")
    parser.add_argument("--env", required=True, help="Environment name (dev, staging, production)")
    parser.add_argument("--env-dir", default="environments", help="Directory with env JSON files")
    parser.add_argument("--output-dir", default="generated", help="Output directory")
    args = parser.parse_args()
    
    generate_config(args.template, args.env, args.env_dir, args.output_dir)
```

### Pattern 5: Parallel Execution (Run Tasks Concurrently)

```python
from concurrent.futures import ThreadPoolExecutor, as_completed

def check_server(host):
    """Check a single server's health."""
    import subprocess
    result = subprocess.run(
        ["ping", "-c", "1", "-W", "2", host],
        capture_output=True
    )
    return host, result.returncode == 0

# Check 10 servers simultaneously (not one-by-one!)
servers = ["192.168.1.10", "192.168.1.11", "192.168.1.12", "192.168.1.13"]

with ThreadPoolExecutor(max_workers=10) as executor:
    futures = {executor.submit(check_server, s): s for s in servers}
    
    for future in as_completed(futures):
        host, is_up = future.result()
        status = "UP ✅" if is_up else "DOWN ❌"
        print(f"  {host:>18} — {status}")
```

### Pattern 6: Watchdog — Watch Files for Changes

```python
import time
import os

def watch_file(filepath, callback, interval=5):
    """Watch a file for changes and run callback when modified."""
    last_mtime = os.path.getmtime(filepath)
    
    print(f"👁️  Watching: {filepath}")
    
    while True:
        current_mtime = os.path.getmtime(filepath)
        
        if current_mtime != last_mtime:
            print(f"📝 File changed! ({filepath})")
            callback(filepath)
            last_mtime = current_mtime
        
        time.sleep(interval)
```

### Key Takeaways:

1. **Always use `argparse`** — makes your scripts flexible and self-documenting
2. **Always use `logging`** — not `print`, because you need timestamps and log files
3. **Always add `--dry-run`** — test before you break things
4. **Always handle errors** — your scripts run unattended, so failures must be clear
5. **Always use timeouts** — nothing should hang forever
6. **Always add retry logic** — networks and services fail sometimes
7. **Use `subprocess.run()`, not `os.system()`** — it's safer and gives you output control
