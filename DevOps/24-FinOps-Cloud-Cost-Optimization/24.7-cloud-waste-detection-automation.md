## 1. What is Cloud Waste?

**Cloud waste** is any cloud resource you're **paying for but not using** (or not using efficiently).

Think of it like this:

> 💡 You leave the lights on in rooms you're not in, leave the AC running when nobody's home, and keep paying for a gym membership you haven't used in 6 months. That's cloud waste — but with servers, databases, and storage volumes.

### Types of Cloud Waste

```
Cloud Waste
├── 🔴 Completely Idle Resources
│   ├── Stopped EC2 instances (still paying for EBS)
│   ├── Unattached EBS volumes
│   ├── Unused Elastic IPs
│   └── Empty load balancers
│
├── 🟡 Under-Utilized Resources
│   ├── EC2 with 5% CPU usage
│   ├── RDS with 10% connections used
│   └── Over-provisioned Lambda memory
│
├── 🟠 Forgotten Resources
│   ├── Old EBS snapshots (months/years old)
│   ├── Stale AMIs
│   ├── Expired SSL certificates
│   └── Test/dev resources left running
│
└── 🔵 Architectural Waste
    ├── NAT Gateway for internal-only traffic
    ├── Multi-AZ on dev databases
    ├── Provisioned IOPS nobody uses
    └── Premium storage tier for cold data
```

## 4. AWS Waste Detection — Manual Methods

Let's go through how to find each type of waste on AWS using the CLI.

### a) Find Unattached EBS Volumes

These are volumes that exist but aren't attached to any EC2 instance. You're still paying for them!

```bash
# List all unattached EBS volumes
aws ec2 describe-volumes \
  --filters "Name=status,Values=available" \
  --query "Volumes[*].{ID:VolumeId, Size:Size, Type:VolumeType, Created:CreateTime}" \
  --output table

# Example output:
# ┌──────────────────────┬────────┬──────────┬───────────────────────┐
# │         ID           │  Size  │   Type   │       Created         │
# ├──────────────────────┼────────┼──────────┼───────────────────────┤
# │ vol-0a1b2c3d4e5f6    │   100  │   gp3    │ 2023-06-15T10:30:00Z  │
# │ vol-0x9y8z7w6v5u4    │   500  │   gp2    │ 2023-01-20T08:15:00Z  │
# │ vol-0m3n4o5p6q7r8    │    50  │   gp3    │ 2024-03-05T14:45:00Z  │
# └──────────────────────┴────────┴──────────┴───────────────────────┘

# Calculate total wasted storage
aws ec2 describe-volumes \
  --filters "Name=status,Values=available" \
  --query "sum(Volumes[].Size)" \
  --output text
# Output: 650 (GB)
# At $0.08/GB for gp3, that's $52/month wasted!
```

### b) Find Old EBS Snapshots

```bash
# Find snapshots older than 90 days
aws ec2 describe-snapshots \
  --owner-ids self \
  --query "Snapshots[?StartTime<='$(date -d '-90 days' --iso-8601)'].{ID:SnapshotId, Size:VolumeSize, Started:StartTime, Description:Description}" \
  --output table

# Count total snapshots and their cost
aws ec2 describe-snapshots \
  --owner-ids self \
  --query "length(Snapshots[])"
# If you have 500 snapshots averaging 50GB each:
# 500 × 50GB × $0.05/GB = $1,250/month!
```

### c) Find Unused Elastic IPs

```bash
# Find Elastic IPs that are not associated with any instance
aws ec2 describe-addresses \
  --query "Addresses[?AssociationId==null].{IP:PublicIp, AllocationId:AllocationId}" \
  --output table

# Each unused EIP costs $3.60/month
```

### d) Find Idle Load Balancers

```bash
# Find ALBs with no targets
for alb_arn in $(aws elbv2 describe-load-balancers --query "LoadBalancers[].LoadBalancerArn" --output text); do
  target_groups=$(aws elbv2 describe-target-groups \
    --load-balancer-arn "$alb_arn" \
    --query "TargetGroups[].TargetGroupArn" --output text)
  
  has_targets=false
  for tg_arn in $target_groups; do
    targets=$(aws elbv2 describe-target-health \
      --target-group-arn "$tg_arn" \
      --query "length(TargetHealthDescriptions[])" --output text)
    if [ "$targets" -gt 0 ]; then
      has_targets=true
    fi
  done
  
  if [ "$has_targets" = false ]; then
    alb_name=$(aws elbv2 describe-load-balancers \
      --load-balancer-arns "$alb_arn" \
      --query "LoadBalancers[0].LoadBalancerName" --output text)
    echo "IDLE ALB: $alb_name ($alb_arn)"
  fi
done
```

### e) Find Idle EC2 Instances

```bash
# Find EC2 instances with average CPU < 5% over the last 14 days
for instance_id in $(aws ec2 describe-instances \
  --filters "Name=instance-state-name,Values=running" \
  --query "Reservations[].Instances[].InstanceId" --output text); do
  
  avg_cpu=$(aws cloudwatch get-metric-statistics \
    --namespace AWS/EC2 \
    --metric-name CPUUtilization \
    --dimensions "Name=InstanceId,Value=$instance_id" \
    --start-time "$(date -d '-14 days' --iso-8601)T00:00:00Z" \
    --end-time "$(date --iso-8601)T00:00:00Z" \
    --period 86400 \
    --statistics Average \
    --query "average(Datapoints[].Average)" --output text)
  
  if (( $(echo "$avg_cpu < 5" | bc -l) )); then
    instance_type=$(aws ec2 describe-instances \
      --instance-ids "$instance_id" \
      --query "Reservations[0].Instances[0].InstanceType" --output text)
    echo "IDLE: $instance_id ($instance_type) - Avg CPU: ${avg_cpu}%"
  fi
done
```

### f) Find Idle RDS Instances

```bash
# Find RDS instances with < 5% average CPU
for db_instance in $(aws rds describe-db-instances \
  --query "DBInstances[].DBInstanceIdentifier" --output text); do
  
  avg_cpu=$(aws cloudwatch get-metric-statistics \
    --namespace AWS/RDS \
    --metric-name CPUUtilization \
    --dimensions "Name=DBInstanceIdentifier,Value=$db_instance" \
    --start-time "$(date -d '-14 days' --iso-8601)T00:00:00Z" \
    --end-time "$(date --iso-8601)T00:00:00Z" \
    --period 86400 \
    --statistics Average \
    --query "average(Datapoints[].Average)" --output text)
  
  connections=$(aws cloudwatch get-metric-statistics \
    --namespace AWS/RDS \
    --metric-name DatabaseConnections \
    --dimensions "Name=DBInstanceIdentifier,Value=$db_instance" \
    --start-time "$(date -d '-14 days' --iso-8601)T00:00:00Z" \
    --end-time "$(date --iso-8601)T00:00:00Z" \
    --period 86400 \
    --statistics Maximum \
    --query "max(Datapoints[].Maximum)" --output text)
  
  echo "$db_instance - Avg CPU: ${avg_cpu}% - Max Connections: $connections"
done
```

## 7. Automated Waste Detection with Scripts

### Comprehensive AWS Waste Detection Script

Here's a complete Python script that finds ALL types of waste in your AWS account:

```python
#!/usr/bin/env python3
"""
AWS Cloud Waste Detector
Scans your AWS account for unused/wasted resources and generates a report.
"""

import boto3
import json
from datetime import datetime, timedelta
from collections import defaultdict

class AWSWasteDetector:
    def __init__(self, region='us-east-1'):
        self.region = region
        self.ec2 = boto3.client('ec2', region_name=region)
        self.rds = boto3.client('rds', region_name=region)
        self.elb = boto3.client('elbv2', region_name=region)
        self.cloudwatch = boto3.client('cloudwatch', region_name=region)
        self.s3 = boto3.client('s3', region_name=region)
        self.waste_report = defaultdict(list)
        self.total_monthly_waste = 0.0

    def find_unattached_ebs_volumes(self):
        """Find EBS volumes not attached to any EC2 instance."""
        print("🔍 Scanning for unattached EBS volumes...")
        
        volumes = self.ec2.describe_volumes(
            Filters=[{'Name': 'status', 'Values': ['available']}]
        )['Volumes']
        
        monthly_waste = 0
        for vol in volumes:
            # Estimate cost based on volume type
            size = vol['Size']
            vol_type = vol['VolumeType']
            
            price_per_gb = {
                'gp3': 0.08, 'gp2': 0.10, 'io1': 0.125,
                'io2': 0.125, 'st1': 0.045, 'sc1': 0.015,
                'standard': 0.05
            }
            
            cost = size * price_per_gb.get(vol_type, 0.10)
            monthly_waste += cost
            
            self.waste_report['Unattached EBS Volumes'].append({
                'ResourceId': vol['VolumeId'],
                'Size': f"{size} GB",
                'Type': vol_type,
                'Created': vol['CreateTime'].strftime('%Y-%m-%d'),
                'MonthlyCost': f"${cost:.2f}"
            })
        
        self.total_monthly_waste += monthly_waste
        print(f"   Found {len(volumes)} unattached volumes (${monthly_waste:.2f}/month)")

    def find_unused_elastic_ips(self):
        """Find Elastic IPs not associated with any instance."""
        print("🔍 Scanning for unused Elastic IPs...")
        
        addresses = self.ec2.describe_addresses()['Addresses']
        unused = [addr for addr in addresses if 'AssociationId' not in addr]
        
        monthly_waste = len(unused) * 3.60  # $3.60/month per unused EIP
        self.total_monthly_waste += monthly_waste
        
        for addr in unused:
            self.waste_report['Unused Elastic IPs'].append({
                'ResourceId': addr['AllocationId'],
                'PublicIP': addr.get('PublicIp', 'N/A'),
                'MonthlyCost': "$3.60"
            })
        
        print(f"   Found {len(unused)} unused Elastic IPs (${monthly_waste:.2f}/month)")

    def find_old_snapshots(self, days_old=90):
        """Find EBS snapshots older than specified days."""
        print(f"🔍 Scanning for snapshots older than {days_old} days...")
        
        cutoff = datetime.utcnow() - timedelta(days=days_old)
        
        snapshots = self.ec2.describe_snapshots(OwnerId=['self'])['Snapshots']
        old_snapshots = [
            s for s in snapshots
            if s['StartTime'].replace(tzinfo=None) < cutoff
        ]
        
        total_size = sum(s['VolumeSize'] for s in old_snapshots)
        monthly_waste = total_size * 0.05  # $0.05/GB for snapshots
        self.total_monthly_waste += monthly_waste
        
        for snap in old_snapshots:
            self.waste_report['Old Snapshots'].append({
                'ResourceId': snap['SnapshotId'],
                'Size': f"{snap['VolumeSize']} GB",
                'Created': snap['StartTime'].strftime('%Y-%m-%d'),
                'Description': snap.get('Description', 'N/A')[:50],
                'MonthlyCost': f"${snap['VolumeSize'] * 0.05:.2f}"
            })
        
        print(f"   Found {len(old_snapshots)} old snapshots ({total_size} GB = ${monthly_waste:.2f}/month)")

    def find_idle_ec2_instances(self, cpu_threshold=5, days=14):
        """Find EC2 instances with very low CPU utilization."""
        print(f"🔍 Scanning for EC2 instances with avg CPU < {cpu_threshold}%...")
        
        instances = self.ec2.describe_instances(
            Filters=[{'Name': 'instance-state-name', 'Values': ['running']}]
        )
        
        idle_count = 0
        for reservation in instances['Reservations']:
            for instance in reservation['Instances']:
                instance_id = instance['InstanceId']
                
                response = self.cloudwatch.get_metric_statistics(
                    Namespace='AWS/EC2',
                    MetricName='CPUUtilization',
                    Dimensions=[{'Name': 'InstanceId', 'Value': instance_id}],
                    StartTime=datetime.utcnow() - timedelta(days=days),
                    EndTime=datetime.utcnow(),
                    Period=86400,
                    Statistics=['Average']
                )
                
                if response['Datapoints']:
                    avg_cpu = sum(dp['Average'] for dp in response['Datapoints']) / len(response['Datapoints'])
                    
                    if avg_cpu < cpu_threshold:
                        name = 'No Name'
                        for tag in instance.get('Tags', []):
                            if tag['Key'] == 'Name':
                                name = tag['Value']
                        
                        idle_count += 1
                        self.waste_report['Idle EC2 Instances'].append({
                            'ResourceId': instance_id,
                            'Name': name,
                            'Type': instance['InstanceType'],
                            'AvgCPU': f"{avg_cpu:.1f}%",
                            'Note': 'Consider stopping or downsizing'
                        })
        
        print(f"   Found {idle_count} idle EC2 instances")

    def find_idle_load_balancers(self):
        """Find load balancers with no healthy targets."""
        print("🔍 Scanning for idle load balancers...")
        
        lbs = self.elb.describe_load_balancers()['LoadBalancers']
        idle_count = 0
        
        for lb in lbs:
            target_groups = self.elb.describe_target_groups(
                LoadBalancerArn=lb['LoadBalancerArn']
            )['TargetGroups']
            
            has_healthy = False
            for tg in target_groups:
                health = self.elb.describe_target_health(
                    TargetGroupArn=tg['TargetGroupArn']
                )['TargetHealthDescriptions']
                
                if any(t['TargetHealth']['State'] == 'healthy' for t in health):
                    has_healthy = True
                    break
            
            if not has_healthy:
                idle_count += 1
                cost = 22.27  # Approx ALB cost
                self.total_monthly_waste += cost
                
                self.waste_report['Idle Load Balancers'].append({
                    'ResourceId': lb['LoadBalancerName'],
                    'Type': lb['Type'],
                    'DNSName': lb['DNSName'],
                    'MonthlyCost': f"${cost:.2f}"
                })
        
        print(f"   Found {idle_count} idle load balancers")

    def generate_report(self):
        """Generate the final waste report."""
        print("\n" + "=" * 70)
        print("☁️  AWS CLOUD WASTE REPORT")
        print("=" * 70)
        print(f"Region: {self.region}")
        print(f"Scan Date: {datetime.utcnow().strftime('%Y-%m-%d %H:%M UTC')}")
        print(f"Total Estimated Monthly Waste: ${self.total_monthly_waste:.2f}")
        print(f"Estimated Annual Waste: ${self.total_monthly_waste * 12:.2f}")
        print("=" * 70)
        
        for category, items in self.waste_report.items():
            print(f"\n📦 {category} ({len(items)} found)")
            print("-" * 50)
            for item in items[:10]:  # Show top 10
                for key, value in item.items():
                    print(f"   {key}: {value}")
                print()
        
        return {
            'region': self.region,
            'scan_date': datetime.utcnow().isoformat(),
            'total_monthly_waste': self.total_monthly_waste,
            'total_annual_waste': self.total_monthly_waste * 12,
            'categories': dict(self.waste_report)
        }

def main():
    detector = AWSWasteDetector(region='us-east-1')
    
    # Run all scans
    detector.find_unattached_ebs_volumes()
    detector.find_unused_elastic_ips()
    detector.find_old_snapshots(days_old=90)
    detector.find_idle_ec2_instances(cpu_threshold=5)
    detector.find_idle_load_balancers()
    
    # Generate report
    report = detector.generate_report()
    
    # Save JSON report
    with open('waste-report.json', 'w') as f:
        json.dump(report, f, indent=2, default=str)
    
    print(f"\n📄 Full report saved to waste-report.json")

if __name__ == "__main__":
    main()
```

```sql
-- Find unattached EBS volumes with cost estimate
SELECT
  volume_id,
  size,
  volume_type,
  create_time,
  CASE volume_type
    WHEN 'gp3' THEN size * 0.08
    WHEN 'gp2' THEN size * 0.10
    WHEN 'io1' THEN size * 0.125
    ELSE size * 0.10
  END AS monthly_cost_usd
FROM
  aws_ebs_volume
WHERE
  jsonb_array_length(attachments) = 0
ORDER BY
  monthly_cost_usd DESC;

-- Find EC2 instances not used in the last 14 days (low CPU)
SELECT
  i.instance_id,
  i.instance_type,
  i.tags ->> 'Name' AS name,
  m.average AS avg_cpu_14d
FROM
  aws_ec2_instance i
LEFT JOIN
  aws_cloudwatch_metric_statistic m
ON
  m.namespace = 'AWS/EC2'
  AND m.metric_name = 'CPUUtilization'
  AND m.dimensions = jsonb_build_array(
    jsonb_build_object('Name', 'InstanceId', 'Value', i.instance_id)
  )
  AND m.period = 1209600  -- 14 days in seconds
WHERE
  i.instance_state = 'running'
  AND m.average < 5;
```

## 9. Building a Cloud Waste Detection Bot (AWS)

Let's build a complete **Slack bot** that sends weekly waste reports.

### Architecture


### Lambda Function

```python
import boto3
import json
import urllib.request
from datetime import datetime, timedelta
from collections import defaultdict

SLACK_WEBHOOK_URL = "https://hooks.slack.com/services/YOUR/WEBHOOK/URL"
# In production, use AWS Secrets Manager instead of hardcoding!

def scan_waste(region='us-east-1'):
    """Scan for all types of cloud waste."""
    ec2 = boto3.client('ec2', region_name=region)
    findings = []
    total_waste = 0.0
    
    # 1. Unattached EBS volumes
    volumes = ec2.describe_volumes(
        Filters=[{'Name': 'status', 'Values': ['available']}]
    )['Volumes']
    
    vol_waste = sum(v['Size'] * 0.08 for v in volumes)  # Simplified pricing
    total_waste += vol_waste
    if volumes:
        findings.append(f"📦 *{len(volumes)} unattached EBS volumes* — ${vol_waste:.2f}/month")
    
    # 2. Unused Elastic IPs
    addresses = ec2.describe_addresses()['Addresses']
    unused_eips = [a for a in addresses if 'AssociationId' not in a]
    eip_waste = len(unused_eips) * 3.60
    total_waste += eip_waste
    if unused_eips:
        findings.append(f"🌐 *{len(unused_eips)} unused Elastic IPs* — ${eip_waste:.2f}/month")
    
    # 3. Old snapshots (>90 days)
    cutoff = datetime.utcnow() - timedelta(days=90)
    snapshots = ec2.describe_snapshots(OwnerId=['self'])['Snapshots']
    old_snaps = [s for s in snapshots if s['StartTime'].replace(tzinfo=None) < cutoff]
    snap_size = sum(s['VolumeSize'] for s in old_snaps)
    snap_waste = snap_size * 0.05
    total_waste += snap_waste
    if old_snaps:
        findings.append(f"📸 *{len(old_snaps)} old snapshots* ({snap_size}GB) — ${snap_waste:.2f}/month")
    
    return findings, total_waste

def send_slack_report(findings, total_waste):
    """Send waste report to Slack."""
    if not findings:
        message = "✅ *Weekly Waste Scan*: No significant waste detected! Great job! 🎉"
    else:
        lines = [
            "🚨 *Weekly Cloud Waste Report*",
            f"_Scan Date: {datetime.utcnow().strftime('%Y-%m-%d')}_",
            "",
            *findings,
            "",
            f"💰 *Total Monthly Waste: ${total_waste:.2f}*",
            f"💰 *Projected Annual Waste: ${total_waste * 12:.2f}*",
            "",
            "_Review and clean up to save money!_"
        ]
        message = "\n".join(lines)
    
    payload = json.dumps({"text": message}).encode('utf-8')
    req = urllib.request.Request(
        SLACK_WEBHOOK_URL,
        data=payload,
        headers={'Content-Type': 'application/json'}
    )
    urllib.request.urlopen(req)

def lambda_handler(event, context):
    """Main Lambda handler."""
    findings, total_waste = scan_waste()
    send_slack_report(findings, total_waste)
    
    return {
        'statusCode': 200,
        'body': json.dumps({
            'findings': len(findings),
            'total_monthly_waste': total_waste
        })
    }
```

## 10. Automating Cleanup with Lambda & EventBridge

### Auto-Delete Unattached EBS Volumes (After Warning Period)

```python
import boto3
from datetime import datetime, timedelta

def lambda_handler(event, context):
    """
    Two-phase cleanup for unattached EBS volumes:
    Phase 1: Tag volumes that have been unattached for 7+ days
    Phase 2: Delete volumes that were tagged 7+ days ago (14 days total)
    """
    ec2 = boto3.client('ec2')
    
    # Phase 1: Tag newly detected unattached volumes
    unattached = ec2.describe_volumes(
        Filters=[
            {'Name': 'status', 'Values': ['available']},
            {'Name': 'tag-key', 'Values': ['waste-detected']},  # NOT already tagged
        ]
    )['Volumes']
    
    # Find volumes without the waste tag
    all_unattached = ec2.describe_volumes(
        Filters=[{'Name': 'status', 'Values': ['available']}]
    )['Volumes']
    
    tagged_ids = {v['VolumeId'] for v in unattached}
    
    for vol in all_unattached:
        if vol['VolumeId'] not in tagged_ids:
            # Check if it has a "keep" tag
            tags = {t['Key']: t['Value'] for t in vol.get('Tags', [])}
            if 'keep' not in tags and 'waste-detected' not in tags:
                ec2.create_tags(
                    Resources=[vol['VolumeId']],
                    Tags=[
                        {'Key': 'waste-detected', 'Value': datetime.utcnow().strftime('%Y-%m-%d')},
                        {'Key': 'waste-action', 'Value': 'scheduled-delete'}
                    ]
                )
                print(f"Tagged: {vol['VolumeId']} for deletion")
    
    # Phase 2: Delete volumes tagged more than 7 days ago
    cutoff = (datetime.utcnow() - timedelta(days=7)).strftime('%Y-%m-%d')
    
    for vol in unattached:
        tags = {t['Key']: t['Value'] for t in vol.get('Tags', [])}
        waste_date = tags.get('waste-detected', '')
        
        if waste_date and waste_date <= cutoff:
            # Create a snapshot before deleting (safety net)
            ec2.create_snapshot(
                VolumeId=vol['VolumeId'],
                Description=f"Auto-backup before waste cleanup on {datetime.utcnow().strftime('%Y-%m-%d')}",
                TagSpecifications=[{
                    'ResourceType': 'snapshot',
                    'Tags': [
                        {'Key': 'created-by', 'Value': 'waste-cleanup'},
                        {'Key': 'original-volume', 'Value': vol['VolumeId']},
                        {'Key': 'auto-delete-after', 'Value': (datetime.utcnow() + timedelta(days=30)).strftime('%Y-%m-%d')}
                    ]
                }]
            )
            
            # Delete the volume
            ec2.delete_volume(VolumeId=vol['VolumeId'])
            print(f"Deleted: {vol['VolumeId']} (was unattached since {waste_date})")
    
    return {'statusCode': 200}
```

### Why the Two-Phase Approach?

```
Day 1:  Volume becomes unattached
Day 7:  Lambda tags it as "waste-detected" 🏷️
Day 14: Lambda snapshots it + deletes it 🗑️
Day 44: Auto-delete the backup snapshot 📸

This gives the owner 14 days to re-attach it!
```

## 13. Governance & Tagging Policies

### Why Tagging is Critical

Without tags, you can't:
- Know who owns a resource (who to ask before deleting)
- Know which project a resource belongs to (for cost allocation)
- Know if a resource is safe to delete (is it production or dev?)
- Automate cleanup (which resources are OK to stop/delete?)

### Required Tags

Every organization should enforce these minimum tags:

| Tag Key | Purpose | Example |
|---|---|---|
| `Name` | Human-readable name | `web-server-prod-1` |
| `Environment` | prod/staging/dev/test | `production` |
| `Owner` | Team or person responsible | `platform-team` |
| `Project` | Which project | `checkout-v2` |
| `CostCenter` | Finance department code | `CC-1234` |
| `ManagedBy` | How it was created | `terraform` |
| `Expiry` | When it should be reviewed/deleted | `2024-12-31` |

### Building a Simple Anomaly Detector

```python
import boto3
from datetime import datetime, timedelta

def check_cost_anomaly():
    """Compare today's spend with the 7-day average."""
    ce = boto3.client('ce')
    
    today = datetime.utcnow().strftime('%Y-%m-%d')
    yesterday = (datetime.utcnow() - timedelta(days=1)).strftime('%Y-%m-%d')
    week_ago = (datetime.utcnow() - timedelta(days=8)).strftime('%Y-%m-%d')
    
    # Get yesterday's cost
    yesterday_cost = ce.get_cost_and_usage(
        TimePeriod={'Start': yesterday, 'End': today},
        Granularity='DAILY',
        Metrics=['UnblendedCost']
    )
    daily_cost = float(
        yesterday_cost['ResultsByTime'][0]['Total']['UnblendedCost']['Amount']
    )
    
    # Get 7-day average
    weekly_cost = ce.get_cost_and_usage(
        TimePeriod={'Start': week_ago, 'End': yesterday},
        Granularity='DAILY',
        Metrics=['UnblendedCost']
    )
    
    daily_costs = [
        float(day['Total']['UnblendedCost']['Amount'])
        for day in weekly_cost['ResultsByTime']
    ]
    avg_daily = sum(daily_costs) / len(daily_costs) if daily_costs else 0
    
    # Check for anomaly (>50% above average)
    if avg_daily > 0 and daily_cost > avg_daily * 1.5:
        percentage = ((daily_cost - avg_daily) / avg_daily) * 100
        print(f"🚨 ANOMALY: Yesterday's cost (${daily_cost:.2f}) is {percentage:.0f}% above the 7-day average (${avg_daily:.2f})")
        
        # Get cost breakdown by service
        breakdown = ce.get_cost_and_usage(
            TimePeriod={'Start': yesterday, 'End': today},
            Granularity='DAILY',
            Metrics=['UnblendedCost'],
            GroupBy=[{'Type': 'DIMENSION', 'Key': 'SERVICE'}]
        )
        
        print("\nTop services:")
        services = breakdown['ResultsByTime'][0]['Groups']
        services.sort(key=lambda x: float(x['Metrics']['UnblendedCost']['Amount']), reverse=True)
        
        for svc in services[:5]:
            cost = float(svc['Metrics']['UnblendedCost']['Amount'])
            print(f"  {svc['Keys'][0]}: ${cost:.2f}")
        
        return True  # Anomaly detected
    
    print(f"✅ Normal: Yesterday ${daily_cost:.2f} vs 7-day avg ${avg_daily:.2f}")
    return False

if __name__ == "__main__":
    check_cost_anomaly()
```
